2017暑假总结

大三进大四的暑假已经过完了。

也就意味着大学真的已经结束了。也不是我无病呻吟，只是想发几句感叹罢了。

这个暑假自己也知道要坚定信念，专心复习考研，看样子自己也确实是没有辜负自己当初的本愿。
但是这还远远不够。

最终自己还是选择了自己的原本专业控制原理，并且想往模式识别方向发展。

说一下自己目前的进度：

  1. 数学
    大致来说已经将高等数学过了一遍，知道了最基本的套路，还需要大量做题才能巩固基础。
  2. 英语
    进度很慢，而且很迷茫，在努力学习恋恋有词当中。
  3. 政治
    还没有开始。

  4. 专业课
    我选择了自动控制原理作为专业书籍，现在正在跟着石群的视频教程学习，效果挺不错的，目前将时域部分已经看完。

    确实，这个进度真的挺慢的，但我还在努力ing…
    说到底，还是想起了辛星当时和我们说的，“不但万不得已千万不要选择考研这条路。”
    现在想来确实是这样。
    考研期间我才刚过一个暑假，期间一个人自觉每天7:40左右起床，晚上9:00左右回到寝室，期间舍友来来回回更替了几个，但是我却一直没有离开。
    一个人就这样度过了大半个月，心中其实很期望有个人可以和我一起分享每天的乐事，但是，
    现实情况不允许我这样。

    自己心中的信念不允许自己后退!
